🧪 Experiment 1: VM vs Containers

🎯 Objective

To understand the difference between Virtual Machines and Containers using Vagrant, VirtualBox, and Docker.

🛠️ Tools Required

  • Vagrant
  • VirtualBox / QEMU
  • WSL
  • Docker

⚙️ Commands Used

vagrant init ubuntu/jammy64
vagrant up
vagrant ssh
sudo apt update
sudo apt install nginx

docker run -d -p 8080:80 nginx
docker ps
docker stop container_id

📌 Procedure

  1. Initialize Vagrant environment
  2. Start virtual machine using vagrant up
  3. Login into VM using vagrant ssh
  4. Install NGINX web server
  5. Run Docker container with NGINX
  6. Compare performance of VM and container

📊 Observations

  • VM takes more time to start
  • Consumes more RAM and CPU
  • Container starts instantly
  • Uses fewer resources

💡 Conclusion

Containers are lightweight, faster, and more efficient compared to Virtual Machines because they share the host OS kernel.
