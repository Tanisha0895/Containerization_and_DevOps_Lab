🧪 Experiment 2: Docker Installation & Basic Commands

🎯 Objective

To understand Docker installation, working of Docker containers, and basic Docker commands.

🛠️ Tools Required

  • Docker
  • WSL / Linux Environment
  • Internet Connection

⚙️ Commands Used

docker --version
docker pull nginx
docker images
docker run -d -p 8080:80 nginx
docker ps
docker stop container_id
docker rm container_id
docker rmi image_id

📌 Procedure

  1. Install Docker in the system
  2. Verify installation using docker --version
  3. Pull nginx image from Docker Hub
  4. Run container using port mapping
  5. Check running containers
  6. Stop and remove container
  7. Remove unused images

📊 Output / Result

Successfully installed Docker and executed basic commands. NGINX container was deployed and accessed via browser using localhost:8080.

💡 Conclusion

Docker simplifies application deployment by packaging applications and dependencies into containers, ensuring consistency across environments.
