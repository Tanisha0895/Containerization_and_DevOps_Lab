🚀 Dockerized Application (My App)

🎯 Objective

To containerize a simple web application using Docker and run it inside a container.

🛠️ Technologies Used

  • Docker
  • Python Flask / Node.js
  • Dockerfile

📂 Project Structure

my-app/
│── app.py / app.js
│── requirements.txt / package.json
│── Dockerfile

⚙️ Dockerfile

FROM python:3.9-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 5000
CMD ["python", "app.py"]

⚙️ Commands Used

1️⃣ Build Image

docker build -t my-app .

2️⃣ Run Container

docker run -d -p 5000:5000 --name my-app-container my-app

3️⃣ Check Running Containers

docker ps

4️⃣ View Logs

docker logs my-app-container

5️⃣ Stop Container

docker stop my-app-container

📌 Procedure

  1. Create application (Flask / Node.js)
  2. Create Dockerfile
  3. Build Docker image
  4. Run container
  5. Access application via browser

📊 Output

Application runs successfully on http://localhost:5000 inside Docker container.

💡 Conclusion

Docker helps in packaging applications with dependencies, making deployment consistent and efficient across environments.
